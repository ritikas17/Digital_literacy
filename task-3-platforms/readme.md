# Task 3: Platforms

## Part A: HackerRank

### What is HackerRank?
HackerRank is an online platform where students and developers can practice coding, solve challenges across multiple programming languages, and earn skill-based certifications. It is widely used by companies for technical hiring, making it a valuable tool for placement preparation. Its short, structured exercises make it easy to stay consistent while steadily improving programming skills.

### Achievement
Successfully completed a Python challenge and earned the **Python Badge** on HackerRank.

## Part B: Google Forms & Sheets — Digital Literacy Awareness Quiz

### About the Quiz
A Digital Literacy Awareness Quiz was created using Google Forms, consisting of five questions designed to test students' understanding of basic digital concepts and safe online behavior.

### The Power of G-Suite Integration
One of the biggest strengths of G-Suite is how seamlessly its applications work together. Once the quiz was created on Google Forms, it was linked directly to Google Sheets, which automatically collected and organized every response in real time — no manual data entry required. This kind of built-in collaboration between tools makes G-Suite incredibly powerful for students and professionals alike, allowing them to collect, analyze, and act on data all within the same ecosystem.
